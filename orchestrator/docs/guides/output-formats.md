# Output Formats

Understanding the output files generated by `color-scheme`.

---

## Output Directory

Default: `~/.config/color-scheme/output/`

Override with `--output-dir`.

---

## Available Formats

| Format | File | Description |
|--------|------|-------------|
| `json` | `colors.json` | Structured JSON data |
| `sh` | `colors.sh` | Shell environment variables |
| `css` | `colors.css` | CSS custom properties |
| `gtk.css` | `gtk.css` | GTK-compatible CSS |
| `yaml` | `colors.yaml` | YAML format |
| `toml` | `colors.toml` | TOML format |
| `sequences` | `sequences` | Terminal escape sequences |
| `rasi` | `colors.rasi` | Rofi theme format |

---

## Selecting Formats

```bash
# Generate only JSON and CSS
uv run color-scheme generate image.png --formats json css

# Generate all formats
uv run color-scheme generate image.png --formats json sh css yaml
```

## Using Generated Files

### In shell scripts

```bash
source ~/.config/color-scheme/output/colors.sh
echo $BACKGROUND
```

### In web projects

```html
<link rel="stylesheet" href="path/to/colors.css">
```

### In rofi

```bash
rofi -theme ~/.config/color-scheme/output/colors.rasi
```
