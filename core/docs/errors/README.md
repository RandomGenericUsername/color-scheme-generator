## Purpose

This section documents significant errors encountered during development or usage, along with their root causes and solutions. This serves as:

- A reference for recurring issues
- Documentation of lessons learned
- A guide for avoiding similar problems

---

## Contributing

When you encounter a significant error:

1. Document the error in `incident-log.md`
2. Include the root cause
3. Document the solution
4. Add any preventive measures

See [Incident Log](incident-log.md) for format.
