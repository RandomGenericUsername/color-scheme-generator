# Output Formats

Understanding the output files generated by `colorscheme-gen`.

---

## Output Directory

Default: `~/.config/color-scheme/output/`

Override with `--output-dir` or in settings.

---

## Available Formats

| Format | File | Description |
|--------|------|-------------|
| `json` | `colors.json` | Structured JSON data |
| `sh` | `colors.sh` | Shell environment variables |
| `css` | `colors.css` | CSS custom properties |
| `gtk.css` | `gtk.css` | GTK-compatible CSS |
| `yaml` | `colors.yaml` | YAML format |
| `toml` | `colors.toml` | TOML format |
| `sequences` | `sequences` | Terminal escape sequences |
| `rasi` | `colors.rasi` | Rofi theme format |

---

## Format Details

### JSON (`colors.json`)

```json
{
  "wallpaper": "/path/to/image.png",
  "special": {
    "background": "#1a1b26",
    "foreground": "#c0caf5",
    "cursor": "#c0caf5"
  },
  "colors": {
    "color0": "#1a1b26",
    "color1": "#f7768e",
    "color2": "#9ece6a",
    "color3": "#e0af68",
    ...
    "color15": "#c0caf5"
  }
}
```

### Shell (`colors.sh`)

```bash
export WALLPAPER="/path/to/image.png"
export BACKGROUND="#1a1b26"
export FOREGROUND="#c0caf5"
export COLOR0="#1a1b26"
export COLOR1="#f7768e"
...
```

Usage:

```bash
source ~/.config/color-scheme/output/colors.sh
echo $BACKGROUND
```

### CSS (`colors.css`)

```css
:root {
    --background: #1a1b26;
    --foreground: #c0caf5;
    --color0: #1a1b26;
    --color1: #f7768e;
    ...
}
```

### YAML (`colors.yaml`)

```yaml
wallpaper: /path/to/image.png
special:
  background: "#1a1b26"
  foreground: "#c0caf5"
colors:
  color0: "#1a1b26"
  color1: "#f7768e"
  ...
```

---

## Selecting Formats

```bash
# Generate only JSON and CSS
uv run colorscheme-gen generate image.png --formats json css

# Generate all formats
uv run colorscheme-gen generate image.png --formats json sh css yaml toml
```

---

## Using Generated Files

### In shell scripts

```bash
source ~/.config/color-scheme/output/colors.sh
# Now use $BACKGROUND, $FOREGROUND, $COLOR0, etc.
```

### In web projects

```html
<link rel="stylesheet" href="~/.config/color-scheme/output/colors.css">
```

### In rofi

```bash
rofi -theme ~/.config/color-scheme/output/colors.rasi
```
